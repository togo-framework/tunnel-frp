# tunnel-frp — docs

frp (Fast Reverse Proxy) driver for the togo `tunnel` subsystem.

- **Install:** `togo install togo-framework/tunnel-frp`
- **Source:** https://github.com/togo-framework/tunnel-frp
- **Base:** https://github.com/togo-framework/tunnel

Requires `frpc` + an frps server. Set `TUNNEL_DRIVER=frp` + the `FRP_*` env.
See the [README](../README.md).
